JavaScript Essential for developers

JS Concept topic wise

  1. Functions: Scope, Clousure and this keyword
  2. ES6: arrow functions, destructuring, spread/rest operators, template literals
  3. Asynchronous JS: Promises and async/await for asynchronous operations
  4. DOM Manipulation: Document Object Model
  5. Event Handling: event listeners, event delegation, event object
  6. Prototypes and Classes: Prototypal inheritance and ES6 classes.
  7. Closures: most asked in interviews
  8. Design Patterns: Singleton, Observer, and Module (most common)
  9. Error handing: Understand try...catch
  10. Data Structures: Arrays, objects, maps, and sets (most important)
  11. Functional Programming: Know about map, filter, and reduce

JavaScript Coding Challenges

Arrays:

  1. Find Maximum Number in Array
    2. Write a function to find the largest number in an array.
  2. Array Sum
    3. Create a function that calculates the sum of all elements in an array.
  3. Remove Duplicates
    4. Implement a function that removes duplicate elements from an array.
  4. Array Rotation
    5. Rotate the elements of an array to the right by a given number of steps.
  5. Find Second Maximum Number
    6. Write a function to find the second largest number in an array.
  6. Array Intersection
    7. Find common elements between two arrays.
  7. Array Chunking
    8. Split an array into chunks of a specified size.
  8. Flatten Nested Arrays
    9. Implement a function to flatten a nested array.
  9. Check for Majority Element
    10. Find the element that appears more than n/2 times in an array.

Objects:

  1. Object Key Count
    2. Write a function that counts the number of keys in an object.
  2. Object Values Sum
    3. Create a function that finds the sum of all values in an object assuming they are numbers.
  3. Object Property Manipulation
    4. Convert object keys into an array and vice versa.
  4. Deep Object Comparison
    5. Compare two objects for equality, including nested objects or arrays.
  5. Object Filtering
    6. Filter an object based on a condition and return a new object with filtered properties.

Strings:

  1. Reverse a String
    2. Implement a function to reverse a string.
  2. Check for Palindrome
    3. Write a function that checks whether a string is a palindrome or not.
  3. Maximum Occurring Character
    4. Find the most frequent character in a string.
  4. Vowel and Consonant Count
    5. Count the number of vowels and consonants in a string.
  5. String Capitalization
    6. Capitalize the first letter of each word in a sentence.
  6. Anagram Check
    7. Check if two strings are anagrams of each other.

Advanced:

  1. Merge Arrays
    2. Write a function to merge two sorted arrays into a single sorted array.
  2. Find Missing Number
    3. Find the missing number in an array of integers from 1 to n.
  3. Rotate Matrix
    4. Rotate a given matrix by 90 degrees clockwise.
  4. Largest Subarray Sum
    5. Find the largest sum of contiguous subarray within a one-dimensional array.
  5. Group Anagrams
    6. Group an array of strings into groups of anagrams.
  6. Implement Stack using Array
    7. Create a stack data structure using arrays.
  7. Implement Queue using Array
    8. Implement a queue data structure using arrays.
  8. Sort Array without 'sort()' Method
    9. Implement a sorting function without using the 'sort()' method.
  9. Shuffle an Array
    10. Write a function to shuffle the elements of an array randomly.
  10. Binary Search in a Sorted Array
    11. Implement a binary search algorithm to find an element in a sorted array.
